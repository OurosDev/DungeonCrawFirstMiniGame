# v0.11.2 — Polish menus et orientation des modèles 3D

Date de préparation : 2026-06-19

## Objectif

`v0.11.2` est une release de polish ciblée.

Elle améliore :
- l'écran titre / menu principal ;
- l'écran de création d'équipe ;
- la compréhension des boutons de roll ;
- l'orientation des modèles 3D spéciaux du donjon.

Cette version ne modifie pas le gameplay, les sauvegardes, les layouts ni le contenu de progression.

## Ajouts principaux

### Menu principal

- Boutons du menu principal texturés avec le style UI NineSlice.
- Grand cadre autour de `DONJON DES SERPENTS` retiré.
- Menu principal rendu plus sobre : fond sombre, titre texte, boutons texturés.
- Panneau Options texturé.

### Création d'équipe

- Cadre principal texturé.
- Boutons principaux texturés.
- Boutons de classe texturés.
- Cohérence visuelle améliorée avec le reste de l'UI `v0.11+`.

### Aide contextuelle des boutons de roll

Ajout d'une fenêtre flottante près du curseur au survol souris de :
- `Relancer`
- `Stocker`
- `Reprendre`

Textes ajoutés :
```text
Relancer :
Relance les stats affichées.
Le roll actuel est remplacé.

Stocker :
Garde le roll actuel en réserve.
Utile pour comparer avant de choisir.

Reprendre :
Remplace les stats actuelles
par le roll stocké.
```

### Orientation des modèles 3D spéciaux

Règle d'orientation des modèles spéciaux :
1. conserver l'orientation naturelle si elle regarde déjà une case `.`;
2. sinon, chercher une case `.` adjacente ;
3. sinon, chercher une case praticable non-mur ;
4. sinon, conserver l'orientation naturelle.

Modèles concernés :
- `M` = message / stèle ;
- `C` = coffre ;
- `O` = temple ;
- `B` = boutique.


Orientations modifiées :

Étage 1 :
- Message `M` (3, 1) : nord -> ouest
- Coffre `C` (5, 1) : nord -> sud
- Coffre `C` (27, 9) : nord -> ouest
- Coffre `C` (15, 19) : nord -> ouest

Étage 2 :
- Message `M` (1, 1) : nord -> sud
- Coffre `C` (25, 7) : nord -> est
- Coffre `C` (1, 13) : nord -> sud
- Coffre `C` (15, 15) : nord -> ouest
- Message `M` (21, 17) : nord -> ouest

Orientations conservées :

Étage 1 :
- Temple `O` (29, 1) : conservé ouest
- Boutique `B` (15, 15) : conservé ouest
- Message `M` (25, 18) : conservé nord

Étage 2 :
- Boutique `B` (13, 5) : conservé ouest
- Message `M` (3, 11) : conservé nord
- Temple `O` (5, 15) : conservé ouest
