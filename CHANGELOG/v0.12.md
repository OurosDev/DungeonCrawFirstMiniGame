# v0.12 — Équilibrage combat, sort découvert et corrections UI

Date de préparation : 2026-06-22

## Objectif

`v0.12` est une release de progression et d'équilibrage.

Elle rend le sort trouvé à l'étage 1 réellement exploitable en combat, ajuste le coût du sort de base du Mage, augmente la durée des combats contre les monstres normaux, et corrige plusieurs détails d'interface apparus après l'ajout de la police globale.

## Gameplay et combat

### Mage — sort de base

Le coût en mana du sort de base du Mage est doublé :

```text
Étincelle / spark : 3 PM -> 6 PM
```

Les dégâts du sort ne sont pas modifiés.

### Sort découvert à l'étage 1

Le sort `Éclat de givre` peut maintenant être utilisé en combat quand les conditions sont réunies :

```text
sort : Éclat de givre / ice_shard
découverte requise : spell_ice_shard
classe : Mage
niveau requis : 2
coût : 5 PM
```

La découverte est maintenant suivie comme une progression de groupe.

Flux attendu :

```text
découverte dans le donjon
-> GameSession mémorise spell_ice_shard
-> SaveManager sauvegarde la découverte
-> CombatAbilityResolver autorise le sort si le héros est compatible
-> le grimoire de combat peut proposer Éclat de givre
```

### PV des monstres

Les monstres normaux gagnent environ 25 % de PV :

```text
Zombie        12 -> 15
Chauve-souris  8 -> 10
Gobelin       16 -> 20
Troll         34 -> 43
Gardien       45 -> 56
```

Le boss gardien est explicitement exclu de cette hausse :

```text
Gardien boss étage 2 : 225 PV conservés
```

Le boss utilise une base séparée pour éviter toute hausse indirecte liée au gardien normal.

## Sauvegarde

Le format de sauvegarde passe à la version 6.

Nouveau champ sauvegardé :

```text
discovered_ability_ids
```

Compatibilité :

- les anciennes sauvegardes restent chargeables ;
- si le champ n'existe pas, la liste des sorts découverts est initialisée vide ;
- aucune progression existante n'est volontairement supprimée.

## Interface

### Création d'équipe

Les valeurs numériques des rolls sont maintenant colorées :

```text
10      vert
5       jaune
4 ou -  rouge
6 à 9   couleur normale
```

Le total reste en couleur normale.

### Équipement

L'écran `Statut > Équipement` est ajusté :

- le slot `Accessoire` n'est plus caché derrière `Retour statut` ;
- le cadre autour des boutons `Arme / Casque / Armure / Bouclier / Accessoire` est retiré ;
- le cadre autour de `Retour statut` est retiré ;
- l'indication basse inutile est supprimée ;
- les hauteurs et espacements sont réduits pour éviter les chevauchements avec la police globale.

### Journal / Combat

L'interface passe toujours automatiquement sur le canal `Combat` à l'entrée en combat.

En sortie de combat, elle revient désormais automatiquement sur le canal `Journal`.

Les messages de combat restent disponibles dans le canal `Combat`.
