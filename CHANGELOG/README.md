## Règle de maintenance

À chaque release :

1. créer ou mettre à jour le fichier correspondant dans ce dossier ;
2. garder le corps de release complet ou un résumé suffisamment détaillé ;
3. mettre à jour ce `README.md` ;
4. ne pas recopier tout l’historique dans `ROADMAP.md`.
