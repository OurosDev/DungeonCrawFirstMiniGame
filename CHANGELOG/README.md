## Notes

Les versions `v0.x` restent des versions pré-1.0. Le projet ne doit pas être accéléré artificiellement vers `v1.0` tant que la boucle complète, les exports, la documentation et les playtests ne sont pas suffisamment mûrs.

Les builds exportées, logs bruts et sauvegardes locales de testeurs ne doivent pas être poussés dans le dépôt.
