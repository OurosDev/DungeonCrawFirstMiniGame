## Règles

- Chaque release publique doit avoir un fichier `CHANGELOG/vX.Y.md`.
- Les builds exportées ne doivent pas être placées dans le dépôt.
- Les logs bruts de playtest ne doivent pas être ajoutés au changelog.
- Les changelogs doivent rester lisibles par un contributeur qui découvre le projet.
