# v0.11.3 — Fond de menu, polices et lisibilité UI

Date de préparation : 2026-06-22

## Objectif

`v0.11.3` est une release de polish visuel et de lisibilité.

Elle enrichit l'identité du menu principal avec une image de fond et des polices OpenType, puis corrige les régressions de lisibilité provoquées par la nouvelle police globale.

Cette version ne modifie pas :

- le gameplay ;
- les règles de combat ;
- les layouts du donjon ;
- les données d'étage ;
- le format de sauvegarde ;
- les états persistants.

## Ajouts principaux

### Menu principal

- Ajout d'une image de fond dédiée au menu principal.
- Ajout de constantes de layout dans `MainMenu.gd` pour régler facilement :
  - la zone du titre ;
  - la zone des boutons ;
  - la zone d'information ;
  - la zone du panneau Options.
- Ajout d'une police OpenType dédiée au titre `DONJON DES SERPENTS`.
- Conservation du placement ajustable pour adapter les boutons au cadre prévu dans l'image.

### Police globale

- Ajout d'un thème UI global basé sur une police OpenType.
- La police globale est destinée aux textes d'interface du jeu.
- Le titre du menu principal conserve une police dédiée séparée.

Fichiers attendus :

```text
assets/fonts/title_medieval.otf
assets/fonts/game_ui.otf
assets/ui/themes/game_theme.tres
```

Note : les fichiers de police doivent avoir une licence compatible avec leur usage dans le projet public.

### Boutons d'exploration

Les libellés des boutons d'exploration ont été simplifiés pour rester centrés et lisibles avec la nouvelle police.

Avant :

```text
Z / flèche / indication + action
```

Après :

```text
Avancer
Reculer
Gauche
Droite
Carte
Menu
```

Les raccourcis clavier restent inchangés :

```text
Z = avancer
S = reculer
Q = tourner à gauche
D = tourner à droite
E = retour / menu
```

### Carte agrandie et automap

- Correction du tooltip de coordonnées quand `X` ou `Y` passent à deux chiffres.
- Le tooltip utilise maintenant un format compact :

```text
X:12  Y:10
```

- Le label du tooltip force l'absence de retour à la ligne.
- La taille minimale du tooltip a été augmentée pour mieux supporter les polices plus larges.
- La position reste limitée dans le panneau de carte pour réduire les sorties de cadre.

## Points de vigilance

- Vérifier que les polices OpenType sont bien importées par Godot.
- Vérifier la licence des fichiers `.otf` avant de les pousser dans un dépôt public.
- Vérifier que la police globale ne déborde pas dans d'autres écrans à forte densité de texte.
- Garder les builds exportées, logs bruts et sauvegardes locales hors du dépôt.
